## Setting up

Set up the node environment:
```shellsession
$ npm install
```

Set up the `clasp` configuration:
```shellsession
$ clasp clone projectID
```

## Running tests

```shellsession
$ npm run test      # runs unit tests
$ npm run coverage  # displays code coverage results
```

## Pushing changes up

```shellsession
$ clasp push
```
